# EE-ERTTE
This is the framework that achieve efficiency and effectiveness ER-TTE. The framework can be used most mainstream TTE model to achieve this goal.

* Every folders with model name ([SSML](./SSML), [ConSTGAT](./ConSTGAT), [MetaER-TTE](MetaER-TTE),[WDR](./WDR), [WDR-LC](./WDR-LC) [MLPTTE](./MLPTTE), ) can run independently. 
* main.py, FTML.py, [models](./models) and [utills](./utills) are the framework that integrates all models.
